01: What is DOM and what is the purpose of DOM?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree-like model. The purpose of the DOM is to provide a way for programs and scripts to access and manipulate the content and structure of a web document.

02: How will you select HTML elements using DOM? Name the DOM methods.

querySelectorAll()

03:What is event bubble?

Event Bubbling is a concept in the DOM (Document Object Model). It happens when an element receives an event, and that event bubbles up. For example, if you click on a child element within a parent element, the click event will first be triggered on the child element, and then it will propagate up through the parent element, the grandparent element, and so on, until it reaches the root elements .

04: What is a callback function and why will you use it?

A callback function is a function that is passed as an argument to another function and then executed inside the parent function. A common use case for a callback function is in asynchronous programming, where a function must wait for some operation to complete before continuing. Callbacks can also be used in event-driven programming, where a function is triggered in response to some user action, such as clicking a button or scrolling a page.
